Common LDAP Queries

The following examples are all queries built using the LdapQueryBuilder. It assumes you have already loaded the config and have an instance of the LdapManager so you can call buildLdapQuery():

use LdapTools\Configuration;
use LdapTools\LdapManager;

$config = (new Configuration())->load('/path/to/ldap/config.yml');
$ldap = new LdapManager($config);

Get All Groups That a User is a Member Of (Recursively)

This query is Active Directory specific as it uses the matching rule OID IN_CHAIN behind the scenes:

$query = $ldap->buildLdapQuery();

$groups = $query->select()
    ->fromGroups()
    ->where($query->filter()->hasMemberRecursively('ChadSikorra'))
    ->getLdapQuery()
    ->getResult();

Get All Users That Belong to a Group (Recursively)

This query is Active Directory specific as it uses the matching rule OID IN_CHAIN behind the scenes:

$query = $ldap->buildLdapQuery();

$users = $query->select()
    ->fromUsers()
    ->where($query->filter()->isRecursivelyMemberOf('Awesome Employees'))
    ->getLdapQuery()
    ->getResult();
